# Koishi Forum Discourse Reacter

## 简介

这是一个为 Koishi 论坛（<https://forum.koishi.xyz/latest>）设计的用户脚本，通过长按爱心按钮快速为当前帖子的所有可点赞内容点赞。

## 使用方法

1. 安装 Tampermonkey 或其他用户脚本管理器
2. 安装本脚本
3. 访问 Koishi 论坛的任意页面
4. 在页面顶部找到爱心按钮
5. **单击**按钮：跳转到未读帖子页面
6. **长按**按钮：开始为当前帖子所有可点赞内容点赞（爱心变红）
7. **松开**按钮：停止点赞（爱心恢复空心）
