# 网页存储清理器

一个用于清空当前网页所有存储数据的油猴脚本。

## 功能特性

- 📊 **查看存储大小**：显示当前网页的存储数据大小
- 🗑️ **清空所有存储**：一键清空 localStorage、sessionStorage、Cookies、IndexedDB、Cache Storage
- 💾 **导出存储数据**：在清空前可以导出备份存储数据
- 🔄 **自动刷新**：清空后自动刷新页面

## 使用方法

1. 安装脚本后，点击油猴图标
2. 在脚本菜单中会看到三个选项：
   - **📊 存储数据 xxx KB**：点击查看详细信息并选择是否清空
   - **🗑️ 清空所有存储**：直接清空所有存储数据
   - **💾 导出存储数据**：导出当前存储数据为 JSON 文件

## 清理内容

- ✅ localStorage
- ✅ sessionStorage
- ✅ Cookies
- ✅ IndexedDB
- ✅ Cache Storage

## 注意事项

⚠️ **清空操作不可恢复**，请谨慎使用！建议在清空前先使用"导出存储数据"功能备份。

## 适用场景

- 网页出现异常，需要重置所有数据
- 清理网站缓存和存储
- 测试网站的初始状态
- 解决因存储数据导致的问题

## 兼容性

- 支持所有网站（`*://*/*`）
- 需要 Tampermonkey 或其他支持 `GM_registerMenuCommand` 的用户脚本管理器
