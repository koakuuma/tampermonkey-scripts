# ASMR音频声道切换

## 使用场景

- 某些ASMR作品的左右声道可能与你的偏好相反
- 想要体验不同的声场效果
- 需要单独听某一个声道的内容
- 快速切换声道模式来找到最佳听感

## 声道模式

1. **🔊 正常声道**（绿色）- 正常输出
2. **🔄 反转声道**（红色）- 左右声道互换
3. **◀️ 仅左声道**（蓝色）- 混合输出到左声道（右声道静音）
4. **▶️ 仅右声道**（橙色）- 混合输出到右声道（左声道静音）

## 支持的网站

- `https://www.asmr.one/work/*`

## 兼容性

- 支持所有现代浏览器（Chrome、Firefox、Edge、Safari 等）
- 需要浏览器支持 Web Audio API
- 同时支持PC端和移动端页面（www.asmr.one）
