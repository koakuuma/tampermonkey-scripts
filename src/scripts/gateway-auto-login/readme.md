# Gateway Auto Login

这个脚本旨在简化 `172.19.0.1` 网关的登录过程。 它会自动填写用户名、密码并选择运营商，从而节省您的时间和精力。

**注意：**  这个脚本需要您进行配置，但不再需要直接修改脚本代码。

## 安装

1.  安装一个用户脚本管理器，例如 [Tampermonkey](https://www.tampermonkey.net/) (推荐) 或 [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/).
2.  将 `Gateway Auto Login.js` 脚本添加到您的用户脚本管理器中。

## 配置

1.  安装脚本后，打开 `http://172.19.0.1/` 页面。
2.  如果脚本检测到您尚未配置账号信息，将自动显示一个配置界面。
3.  您也可以点击页面右上角的 **“配置自动登录”** 按钮来打开配置界面。
4.  在配置界面中，填写以下信息：
    *   **教职工号/学号：** 您在 `172.19.0.1` 网关上的用户名。
    *   **密码：** 您在 `172.19.0.1` 网关上的密码 (明文显示，方便对照输入)。
    *   **运营商：** 您使用的运营商。从下拉列表中选择正确的运营商。
    *   **输入速度：** 设置用户名和密码的输入速度。可选值：
        *   `1`: 超慢
        *   `2`: 慢
        *   `3`: 中等
        *   `4`: 稍快
        *   `5`: 快
        *   `6`: 极速
    *   **启用自动登录：** 勾选此选项以启用自动登录功能。
    *   **启用调试日志：** 勾选此选项以开启调试日志，方便排查问题 (仅供开发者使用)。
5.  点击 **“保存配置并登录”** 按钮。 脚本会将您的配置保存在浏览器的 `localStorage` 中，下次打开页面时会自动加载。

**重要提示:**  请谨慎处理您的密码信息，虽然密码明文显示，但配置信息保存在您的浏览器本地，请确保您的浏览器安全。

## 使用

1.  确保您已正确安装并配置了脚本。
2.  打开 `http://172.19.0.1/` 页面。
3.  如果启用了自动登录，脚本将自动开始填写表单并尝试登录。

## 注意事项

*   如果脚本无法正常工作，请检查以下事项：
    *   确保您已正确填写了配置信息。
    *   检查 `172.19.0.1` 页面结构是否发生了变化，这可能会导致脚本无法找到正确的表单元素。
    *   如果页面结构发生变化，您可能需要修改脚本中的选择器 (例如 `document.querySelector`) 以匹配新的页面结构。
*   密码明文显示，请在安全的环境下使用。
*   如果您发现任何问题或有任何建议，请随时提出。
